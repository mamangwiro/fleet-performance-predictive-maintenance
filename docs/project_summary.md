# Project Summary

This project focuses on analysing fleet performance data and developing a predictive model to identify vehicles at high risk of breakdown. The objective is to support proactive maintenance planning, reduce downtime, and improve operational efficiency in logistics operations.

Using Python for data processing and machine learning, the project involved data cleaning, feature engineering, exploratory data analysis (EDA), and model development. Classification algorithms such as Logistic Regression and Random Forest were applied to predict breakdown likelihood based on operational variables including mileage, fuel consumption, delivery performance, and maintenance history.

The analysis identified key drivers of breakdown risk and enabled segmentation of the fleet into high, medium, and low-risk categories. A Power BI dashboard was developed to visualise fleet risk exposure, monitor key performance indicators, and support data-driven decision-making.

The solution demonstrates how predictive analytics can be applied in logistics to optimise maintenance strategies, reduce operational costs, and enhance fleet reliability.
