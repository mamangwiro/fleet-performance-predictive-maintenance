# Fleet Predictive Maintenance Model

This project focuses on predicting vehicle breakdown risk using historical fleet data. The goal is to identify high-risk vehicles early and support maintenance planning to reduce downtime and operational costs in logistics operations.

---

## Overview

- **Motivation:** Frequent vehicle breakdowns in logistics operations lead to delays, increased costs, and reduced service reliability.

- **Objective:** To develop a predictive model that identifies vehicles at risk of breakdown using historical operational and maintenance data.

- **Learning Outcomes:** 
  - Applied data cleaning and preprocessing techniques
  - Built predictive models for risk classification
  - Developed data-driven insights for fleet optimisation

---

## Dataset

- **Source:** Simulated fleet dataset based on real logistics operations  
- **Size:** ~500–1000 records  
- **Key Features:**
  - Vehicle ID  
  - Mileage  
  - Maintenance history  
  - Breakdown incidents  
  - Age of vehicle  
  - Fuel usage  

- **Preprocessing:**
  - Handling missing values  
  - Removing duplicates  
  - Feature engineering  

---

## Technologies Used

- Python (Pandas, NumPy, Scikit-learn)  
- Jupyter Notebook  
- Power BI  

---

## Usage

1. Open the main notebook (`analysis.ipynb`)  
2. Run each cell sequentially  
3. Review breakdown risk predictions and visualizations  

---

## Analysis & Visualizations

- Identified key factors influencing vehicle breakdown risk  
- High mileage and irregular maintenance correlate with breakdowns  
- Developed a risk scoring model  

### Key Insights:
- High-mileage vehicles have higher breakdown probability  
- Preventive maintenance reduces failures  
- Risk-based scheduling improves efficiency  

---

## Conclusion

This project demonstrates how predictive analytics can improve fleet management.

### Benefits:
- Reduced downtime  
- Improved maintenance planning  
- Lower operational costs  

### Recommendation:
Adopt predictive maintenance using risk scoring models.
